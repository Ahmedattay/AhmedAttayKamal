## 🚀 Getting Started

1. Clone or download the repository

```bash
git clone https://github.com/Ahmedattay/<repo-name>.git
cd <repo-name>
open index.html
```

## 📱 Responsive Design

Fully responsive across three breakpoints:

| Breakpoint | Target |
|---|---|
| ≤ 1024px | Tablets |
| ≤ 768px | Small tablets & large mobiles |
| ≤ 480px | Mobile phones |
